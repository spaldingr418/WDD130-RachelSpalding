General Experience and Reflections

What was the most challenging part of this project for you? Were you able to overcome it?

Figuring out how to use GitHub with no prior experience after giving birth the Monday that the projects started. Eventually I overcame it by focusing on figuring out how to use it with the desktop application & visual studio while also balancing caring for a newborn.

What did you enjoy most about this project?

Learning how to use it all. I'm so excited to do side projects to learn more on my own.

What would you do differently if you could do the project again?

Spend some time before the projects started to understand what programs I would have to use.

Do you feel more confident in working on real-world projects after this experience? Explain.

Yes, I feel more confident, but I understand I'm a beginner who’s still learning. The biggest factor is that I’m still excited to experiment with projects and continue learning how to create websites via code.
