ICE Challenges

Here are the ICE challenges I have completed this semester and what I learned from each one.

ICE Challenge-Prophet Card

This was the first challenge we completed this semester. During this challenge, the goal was to follow the wireframe model and match it to the html and css code with only a few hints. I learned how to apply the color codes, create a rounded border and how to dig a little deeper to make the wireframe happen.

ICE Challenge-Apostle Spotlight

The goal for this challenge was to use the wireframe provided to apply new features we studied for the week, with minimal information provided. With this lesson I learned how to stick an image as a background, align up specific images as a grid with individual circle borders.
